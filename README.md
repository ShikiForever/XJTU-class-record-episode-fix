# XJTU-New-LMS-Video-Helper (XJTU课程录像分集与批量下载)

> 之前学长针对老版录像平台制作了添加分集和下一集跳转功能的脚本。随着学校更新了新版 LMS 录课平台，原有的脚本已经失效。因此，我在原项目的基础上用AI改进了学长的脚本，让**分集选择**和**下一集跳转**功能适配了新版 LMS 平台，还加入了**录像批量下载/导出下载链接**功能！

## ✨ 核心功能

* **适配新版 LMS 平台**：完全兼容学校最新上线的 LMS 录课系统。
* **无缝分集跳转**：在当前录像界面直接调出播放列表，任意选择某一天的课程，无需退回主页。
* **下一集连播**：看完一集直接点击跳转下一集，沉浸式复习。
* **🚀 [NEW] 批量下载与导出**：
* 支持解析当前课程的视频直链。
* 支持一键批量导出下载链接，方便使用 IDM / Aria2 等下载工具进行多线程满速下载。
* 支持直接在浏览器内单集/批量下载到本地。

## 🛠️ 安装与使用

### 第一步：安装脚本管理器

本脚本需要配合浏览器扩展 **Tampermonkey (油猴)** 使用。如果你尚未安装，请点击下方链接安装：

* [Chrome / Edge 版本](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
* [Firefox 版本](https://addons.mozilla.org/zh-CN/firefox/addon/tampermonkey/)
* [Safari 版本 (Userscripts)](https://apps.apple.com/app/userscripts/id1463298887)

### 第二步：安装脚本

确保油猴插件已开启，点击下方的安装链接即可安装：

**[点击此处安装 XJTU课程录像分集与批量下载](https://greasyfork.org/zh-CN/scripts/585319-xjtu%E8%AF%BE%E7%A8%8B%E5%BD%95%E5%83%8F%E5%88%86%E9%9B%86%E4%B8%8E%E6%89%B9%E9%87%8F%E4%B8%8B%E8%BD%BD)**

> **提示**：点击后会跳转到安装页面，点击 **“安装”** 或 **“Install”** 按钮即可。

## 📖 使用说明

1. 安装完成后，登录并打开学校的 **新版 LMS 录课平台** 播放页面。
2. 等待页面和视频加载，播放器控件附近会出现注入的扩展按钮组（分集、下一集、下载工具）。
3. **跳转功能**：点击“分集”查看全部课时，或点击“下一集”快速切换。
4. **下载功能**：点击“下载/导出”按钮，选择你需要导出的课时，一键复制真实视频链接，或直接触发本地下载。
